# INSE Projects

This repository contains a collection of academic and technical projects completed during my Master’s studies in Information Systems Security.

The goal of this repository is to organize selected coursework projects in a clean, professional, and portfolio-friendly format. Each project demonstrates practical implementation of security, cryptography, networking, or software engineering concepts covered throughout the program.

## Projects

### 1. RSA Encryption, Decryption, and Digital Signature Project

This project is an educational implementation of the RSA cryptosystem completed for an INSE 6110 cryptography assignment.

The project demonstrates:

- Prime number generation
- RSA public and private key calculation
- Modular arithmetic
- Encryption and decryption
- Digital signature generation
- Signature verification
- Square-and-multiply modular exponentiation

The implementation avoids ready-made RSA libraries and focuses on manually implementing the mathematical steps behind RSA.

> Note: This project is for academic learning purposes only and is not suitable for production cryptographic use.
